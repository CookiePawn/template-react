## PROJECT SETTING
```bash
yarn
```


## RUN PROJECT - Scripts
[DEVELOPMENT]
```bash
yarn run start:dev # 개발자 모드 실행
yarn run build:dev # 빌드
```

[PRODUCTION]
```bash
yarn run start:prod # 개발자 모드 실행
yarn run build:prod # 빌드
```

[LOCAL]
```bash
yarn run start:local # 개발자 모드 실행
yarn run build:local # 빌드
```
